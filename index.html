Projetos

Projeto 1

O Central de Estágios é uma plataforma web que conecta estudantes a vagas de estágio e emprego, permitindo o cadastro de currículos, simulação de entrevistas e dicas de carreira. Desenvolvido com HTML5, CSS3, JavaScript no frontend e Flask com SQLAlchemy no backend, o sistema integra perfil de usuário com informações automáticas do currículo, além de permitir a edição de dados adicionais, como foto e habilidades.
